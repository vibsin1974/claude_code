# 네이버 AI 뉴스 검색 스킬

## 사전 요구사항
- 네이버 개발자 센터에서 "뉴스 검색" API 애플리케이션 등록 후 발급
- 환경변수 설정 필요:
  - `NAVER_CLIENT_ID`
  - `NAVER_CLIENT_SECRET`

## 실행 방법

```bash
python3 /home/claude/naver-ai-news/fetch_news.py "인공지능 AI" 5
```

- 인수 1: 검색 키워드 (기본값: `인공지능 AI`)
- 인수 2: 뉴스 개수 (기본값: `5`, 최대 `100`)

## 키워드 가이드

| 사용자 요청 | 권장 키워드 |
|------------|-----------|
| 일반 AI 뉴스 | `인공지능 AI` |
| 생성형 AI | `생성형AI LLM` |
| AI 정책/공공 | `공공기관 인공지능` |
| AI 보안 | `AI 사이버보안` |
| 특정 기업 | `삼성 AI` / `카카오 인공지능` |

## 출력 형식

```
📰 최신 AI 뉴스 [키워드]

1. [기사 제목]
   📍 [날짜]
   💬 [요약]
   🔗 [원문 링크]
```

## API 오류 코드 대응
| 코드 | 원인 | 조치 |
|------|------|------|
| 401 | 인증 실패 | 환경변수 확인 |
| 403 | 권한 없음 | 네이버 API 뉴스 검색 권한 확인 |
| 429 | 호출 한도 초과 | 일일 25,000건 한도 확인 |
